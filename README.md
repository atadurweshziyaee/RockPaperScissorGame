# 🎮✨ Rock Paper Scissors Game 🪨📄✂️

## 🕹️ About the Game
Rock Paper Scissors is a **classic hand game** where you compete against the **computer**. You select **Rock (🪨), Paper (📄), or Scissors (✂️)**, and the computer picks a random choice. The winner is determined based on simple rules, making it an **exciting and quick** game to play! 🎯

---

## 🏆 Game Rules 📝
✅ **Rock (🪨) beats Scissors (✂️)** → Rock crushes Scissors.  
✅ **Scissors (✂️) beats Paper (📄)** → Scissors cut Paper.  
✅ **Paper (📄) beats Rock (🪨)** → Paper covers Rock.  
✅ If both choices are the same, it's a **TIE!** ⚖️

---

## 🎯 How to Play 🎲
1️⃣ **Start the Game:** Open the game in any web browser. 🌐  
2️⃣ **Choose Your Move:** Click **Rock (🪨), Paper (📄), or Scissors (✂️).**  
3️⃣ **Computer Plays:** The computer randomly selects a move. 💻  
4️⃣ **Get Instant Results:** The game determines the winner and displays the outcome. 🏅  
5️⃣ **Replay & Have Fun!** Click the restart button to play again. 🔄
